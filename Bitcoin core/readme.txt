# Bitcoin Core Project

Este proyecto es una implementación de código abierto del cliente de Bitcoin Core.

## Instalación

1. Descarga el código fuente desde [GitHub](https://github.com/bitcoin/bitcoin).
2. Sigue las instrucciones de compilación en el archivo `INSTALL.md`.
3. Una vez compilado, ejecuta el programa `bitcoind` para iniciar el nodo de Bitcoin Core.

## Uso

- Para iniciar el nodo de Bitcoin Core, ejecuta `bitcoind`.
- Para interactuar con la billetera, ejecuta `bitcoin-cli`.
- Para más opciones y comandos, consulta la documentación oficial de [Bitcoin Core](https://bitcoincore.org/en/doc/).
